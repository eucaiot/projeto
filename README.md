# Projeto Exemplo

### Introdução

Este é um exemplo de projeto em equipe (colaborativo) utilizando Git e GitHub. O objetivo é demonstrar como múltiplos colaboradores podem trabalhar juntos em um repositório Git.

### Instruções para Colaboradores

Após clonar o repositório, cada colaborador deve seguir os seguintes passos:

1. **Criar um arquivo** `.env` a partir do arquivo `.env.example`.
2. **Adicionar suas credenciais pessoais** no arquivo `.env` (para desenvolvimento local).

> **NOTA:** Vocês nunca devem preencher seus dados reais no arquivo `.env.example`.

### Credenciais de Exemplo

```env
MYSQL_HOST=localhost
MYSQL_USER=seu_usuario
MYSQL_PASSWORD=sua_senha
MYSQL_DB=seu_banco_de_dados

```
